# SnakeJS
A simple snake game written in HTML, CSS and JS.

## Controls
### PC:
- WASD/Arrow Keys to move
- Space/P to pause
- R to restart
### Mobile:
- Arrow buttons to move
- Double Tap to pause

## How to run
### Online:
Go [here](https://ibrahim-scythe.github.io/SnakeJS/)

### Locally:
Download repository and open `index.html` in any web browser.
